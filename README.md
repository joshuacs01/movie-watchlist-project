# Movie Watchlist App

A full-stack movie watchlist web app built with Node.js and Next.js. Users can create accounts, log in, and manage their personal movie list. The app supports full CRUD functionality, allowing users to add, view, update, and delete movies in their watchlist. It uses JWT for authentication, bcrypt for password security, and a RESTful API built wit Express. All movie data is stored in a MySQL database using Prisma ORM, with protected routes ensuring each user only accesses their own data.

## Features

- User registration and login
- JWT authentication
- Password hashing with bcrypt
- Create and manage movie watchlists
- Mark movies as watched/unwatched
- Protected routes (authentication required)
- REST API built with Express
- MySQL database using Prisma ORM
